What is Security Guard LLM?

Security Guard uses an AI model to analyze user requests before executing the Landing Page Wizard. It detects and blocks potential security threats like:

  • Prompt injection - Attempts to override instructions
  • System info extraction - Requests for server paths, environment variables
  • Malicious code - Scripts that could harm the system
  • Jailbreak attempts - Attempts to bypass safety measures
