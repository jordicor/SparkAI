Security isn't a feature. It's the foundation.

Four independent layers of protection, each unaware of the others' existence. An attacker would need to bypass all of them in sequence.

Cloudflare Edge

DDoS mitigation, Web Application Firewall, bot management, and TLS termination at 300+ global points of presence.

Layer 1

nginx Gateway

Rate limiting (10r/s general, 50r/s static, 5r/s auth), security headers, scanner detection, and static file serving.

Layer 2

Application Middleware

Malicious pattern detection with instant blocking, 404 accumulation tracking, and real IP resolution through proxy chains.

Layer 3

Business Logic

JWT authentication, role-based access control, permission validation, and input sanitization at every endpoint.

Layer 4

Instant Pattern Blocking

WordPress probes, PHP shells, cloud metadata attacks, and path traversal attempts are blocked on first request. 24-hour ban, no exceptions.

Behavioral Detection

10 suspicious 404s in 3 minutes triggers automatic blocking. Legitimate crawlers get more lenience (50 requests) because we recognize them.

Real IP Resolution

We see through proxy chains. CF-Connecting-IP, X-Forwarded-For parsing, direct connection fallback. You can't hide behind proxies.

Our servers don't exist on the public internet.

Zero inbound ports. No exposed IP addresses. Traffic flows through encrypted Cloudflare Tunnels. Attackers can't target what they can't find.

No public IP exposure

Origin servers are invisible to port scanners and reconnaissance tools

Encrypted origin connection

TLS 1.3 between Cloudflare edge and origin through the tunnel

Automatic failover

Tunnel reconnects transparently on network issues without dropping requests

Zero firewall configuration

No ports to open, no rules to maintain, no attack surface to harden

Static content never touches our servers.

Assets are cached at 300+ Cloudflare edge locations. Users download from the nearest node, often in under 20ms.

nginx Local Cache

Static files served directly by nginx. Zero application overhead. Rate-limited at 50 requests/second per IP to prevent abuse.

Cloudflare Edge

30-day browser cache. Automatic WebP conversion. Brotli compression. Images, CSS, JS, and fonts served from the nearest edge node.

Argo Smart Routing

Traffic routed through Cloudflare's private backbone, avoiding congested internet paths. Faster and more reliable than public routing.
